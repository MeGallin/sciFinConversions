Angular 7, Material 7 and Flex-box

Using Angular 7 I wanted to create a practical app that can be used to do some handy calculations. There are two sections, Financial and Science. I will be adding more calculations as time goes on. I will also be adding an invoice component where you would be able to create an voice.

This app was build with angular 7 and styled using angular Material 2 and flex-box. Within the app I made use of routing, Angular's reactive forms and Services. There are two services one the Financial calculations and one for the Scientific calculations.

Future Plans

Going forward I want add an invoice component and also a loggin and registration component using Angular Firebase. I will also be adding some additional features which will enhance the user experience.

Financial

In this section you will be able to calculate payments on a loan, future value of a property, how much you can borrow for a fixed monthly commitment.

Scientific

Here you will able to do basic area conversions or metric conversions and even some basic physics. I will be adding more in the near future.
